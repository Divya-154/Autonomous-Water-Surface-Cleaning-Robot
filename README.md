# Autonomous Water Surface Cleaning Robot

Computer Vision | YOLOv3 | OpenCV | Python | Environmental Monitoring

## Overview

An autonomous water surface monitoring system developed to detect floating waste materials using YOLOv3 and OpenCV. The project integrates object detection, heatmap generation, obstacle analysis, path planning, and a graphical user interface to support intelligent waste identification and monitoring in water bodies.

## Technologies Used

- Python
- YOLOv3
- OpenCV
- NumPy
- Tkinter

## Key Features

- Real-time waste detection
- Heatmap generation
- Path planning algorithms
- Obstacle detection and avoidance
- Multi-view monitoring
- Simulation environment
- GUI-based control dashboard

## Core Modules

- vision.py – Object detection and image processing
- heatmap.py – Heatmap generation
- planner.py – Path planning logic
- motion.py – Movement control logic
- obstacle.py – Obstacle detection
- multiview.py – Multi-camera/view processing
- simulation.py – Simulation environment

## Project Structure

GUI.py serves as the main interface and coordinates all modules for monitoring and autonomous operation.
